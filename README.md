# homebrew-tap

Homebrew tap for [aksisonline/gitswitch](https://github.com/aksisonline/gitswitch).

## Install

```bash
brew install aksisonline/tap/gitswitch
```

Or as two steps:

```bash
brew tap aksisonline/tap
brew install gitswitch
```

Formula is auto-updated by the [gitswitch release workflow](https://github.com/aksisonline/gitswitch/blob/main/.github/workflows/release.yml) on every new release.
